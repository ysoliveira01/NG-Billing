# NG-Billing - Vaga Analista de Produção de TI

# Desafio 1 - Docker

- Escreva um arquivo dockerfile para a construção de uma aplicação python que faça a leitura de um diretório que será mapeado no host local e que retorne a lista de arquivos através de uma chamada HTTP com método GET.
- Descreva o que deverá ser monitorado para que a aplicação se mantenha estável.

# Desafio 2 - Python

- Crie um script python que leia uma base de dados Oracle obtendo o último Id de sequência de uma tabela de controle e o envie por e-mail.

- Descreva como ou faça o script de tal forma que garanta a segurança da senha de acesso à base.

# Desafio 3 - Bash

- Escreva um serviço no linux que monitore um diretório e mova o arquivo instantaneamente para outro diretório quando este for armazenado neste diretório.

- Garanta que o serviço esteja ativo de forma que suba durante o processo de inicialização do linux, caso este venha a ser reiniciado.
